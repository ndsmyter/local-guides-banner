Use on Local Guides Connect

If you want to use on the Local Guides Connect website, you can add this to your signature.

  1. Open Local Guides Connect (make sure you're logged in)
  2. Click on your profile avatar in the top right of the application
  3. Choose "My Profile"
  4. Select "Edit" next to your name
  5. Navigate to the "Personal Information" tab
  6. Copy the text in the text field above to the "Signature" field
  7. Save the profile at the bottom of the page

If you now generate a new comment, you can see the banner in the preview of your message. And of course also after you've saved the comment and go back to the comment.

Generate your own version of the banner, using only 3 easy steps:

  1. Navigate to your Your Contributions page on Google Maps (make sure you're logged in)
  2. You will be redirected to a page with a link resembling: https://www.google.com/maps/contrib/123456789012345678901/contribute/
    which contains the 21-digit ID that is used to create the banner
  3. Copy and paste the whole link to the input field above

Don't just "share" your profile, that will not have the same effect
