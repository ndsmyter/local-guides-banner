Local Guides Banner Generator

Preview

Generate your own version of the banner, using only 3 easy steps:

  1. Navigate to your Your Contributions page on Google Maps (make sure you're logged in)
  2. You will be redirected to a page with a link resembling: https://www.google.com/maps/contrib/123456789012345678901/contribute/
    which contains the 21-digit ID that is used to create the banner
  3. Copy and paste the whole link to the input field above

Don't just "share" your profile, that will not have the same effect
